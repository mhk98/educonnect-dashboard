Feature checklist for EduConnect Dashboard

- Authentication
  - Login page + validation
  - Forgot password flow
  - Create account / registration

- Layout & Navigation
  - Sidebar (collapse/expand)
  - Header (notifications, profile menu, logout)
  - Private routes / auth guard

- Dashboard
  - Dashboard overview cards
  - Dashboard lists / stats / charts
  - Quick links

- Students
  - Students list / table view
  - Student registration (StudentRegister)
  - Student edit profile (StudentEditProfile)
  - Student QR card

- Applications & Enquiries
  - Applications list & details
  - Enquiries management
  - Application-related documents

- Programs & Academics
  - Programs page
  - Program Country / Intake / Years / University / Name management
  - academic feature flows (create/edit)

- Leads & CRM
  - Leads listing and edit
  - Lead documents

- Tasks
  - Task list / task details
  - Task activity / comments

- Notifications
  - Notification dropdown and polling
  - Notification page

- Payments & Wallet
  - Wallet overview
  - CashIn / CashOut flows
  - Commission payments / payment requests / pending payments
  - Payment status UI

- User Management
  - Users listing, filters and role badges
  - Regional managers panel

- Documents
  - Additional documents, EA documents, lead documents handling
  - Upload / view flows

- Notice & Announcements
  - Notices list and details

- UI Components & Patterns
  - Forms page + validation patterns
  - Tables page (sorting/search/pagination)
  - Modals, Buttons, Cards, Typography
  - FilterPanel usage
  - ThemedSuspense / loading states

- Utilities & Integrations
  - Windmill UI theme usage (now fixed to light)
  - Tailwind CSS setup
  - Service worker (PWA) registration
  - LocalStorage usage (token, userId, role, branch, theme)

- Admin / Misc
  - CommissionPayment page
  - Notification settings
  - Reports / charts page
  - Tests feature (src/features/tests)
